# JoblyNest

**Role-Based Job & Task Marketplace (FYP)**

## Project Overview

**JoblyNest** is a web-based job and task marketplace developed as a **Final Year Project (FYP)**.
The platform connects **job posters** with **job seekers** through a secure, role-based system. It enables task posting, job acceptance, admin verification, and online payments.

## Problem Statement

Traditional job platforms lack:

- Proper user verification
- Secure role-based access
- Safe payment mechanisms
- Efficient short-term task matching

JoblyNest addresses these issues by providing a verified, admin-controlled system with secure authentication and payments.

---

## Objectives

- Provide a secure platform for job posting and hiring
- Implement role-based access for different users
- Enable skill and location-based job matching
- Ensure safe online payments
- Maintain data integrity using a structured database

---

## User Roles

- **Job Poster:** Create and manage jobs, hire seekers, make payments
- **Job Seeker:** Create profile, add skills & availability, accept jobs
- **Admin:** Approve users and handle complaints

---

## Key Features

- Role-based authentication
- Admin approval system
- Job posting and acceptance
- Skill and location-based matching
- Availability management
- Secure payments using Stripe
- Responsive user interface

---

## Technology Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database & Auth:** Supabase (PostgreSQL)
- **Payments:** Stripe

---

## System Architecture

```
React Frontend → Node.js/Express → Supabase (PostgreSQL)
                               → Stripe Payment Gateway
```

---

## Conclusion

JoblyNest successfully demonstrates a real-world full-stack application with secure authentication, role-based access control, and online payment integration. The project fulfills the objectives of a Final Year Project and provides a scalable foundation for future enhancements.
